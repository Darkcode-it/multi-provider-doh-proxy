# Docs Directory

Place your screenshot/image files in this directory.

## Required Files

- `preview.png` - Main screenshot of the landing page (recommended size: 1200x600px)

## Image Guidelines

- Format: PNG or JPG
- Recommended size: 1200x600px or larger
- Aspect ratio: 2:1 (width:height)
- File size: Keep under 1MB for GitHub

## How to Add Screenshot

1. Take a screenshot of your landing page
2. Save it as `preview.png` in this directory
3. Commit and push to GitHub

The README.md file will automatically display your screenshot.
